# YTLoader
## This is a program that can download video files and a video sound from YouTube.
### Stack:
* Java SE 17  
* JavaFx
* ffmpeg
* yt-dlp

There are [_ffmpeg_](https://ffmpeg.org/) and [_yt-dlp_](https://github.com/yt-dlp/yt-dlp) documentation.
